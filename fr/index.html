Bienvenue sur Captain Icon

Il était une fois un Jeune homme qui avait une passion pour le dessin depuis l´enfance.

Il voulait partager son talent avec le monde et apporter son petit grain de sable.

Un jour il eut une brillante idée: réaliser un étonnant jeu d´icones gratuites.

Captain Icon sait créer quelque chose de nouveau et de très difficile. C´est pour cela qu´il veut vous aider à éviter ces trois étapes.

Si vous voulez voir le processus de conception de ce jeu d´Icônes gratuites mettez le curseur sur l´image

Traits

  • Icônes Vectorielles

    +350 icônes vectorielles que vous pouvez changer en n´importe quelle taille sans perte de qualité.

  • Catégories

    Un large éventail de catégories, de l´informatique à l´alimentation en passant par d´autres comme le Design, les sports, les réseaux sociaux, le bureau, le climat et bien d´autres catégories.

  • Formats

    Disponibles en EPS, PSD, PING, SVG et Web Font pour que vous puissiez les inclure dans vos projets web, portable et bureau.

  • Icon Fonts

    Utilisez des icones comme Web Font pour créer de belles et claires interfaces pour des sites web et ceci plus facilement.

Personnalisation

Captain Icon a travaillé dur pour fournir un ensemble d´icônes gratuites que vous pouvez personnaliser comme vous voulez. Aussi bien la taille, les couleurs que les formes, les nœuds et bien d´autres choses.

Comme Captain Icon sait exactement comment vous travaillez il a créé le set d´icones gratuites aussi bien pour des applications graphiques que pour des applications vectorielles.
