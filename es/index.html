Bienvenido a Captain Icon

Érase una vez un joven que le apasionaba dibujar desde que era pequeño.

Quería compartir su afición con el mundo y aportar su pequeño granito de arena.

Un día tuvo una brillante idea: realizar un asombroso conjunto de iconos gratis.

Captain Icon sabe que crear algo nuevo es muy difícil. Por eso, quiere ayudarte a evitar estos 3 pasos.

Si quieres ver el proceso de diseño de este conjunto de iconos gratis, pon el cursor sobre la imagen

Características

  • Iconos Vectoriales

    +350 iconos vectoriales que puedes cambiar a cualquier tamaño sin que pierdan calidad.

  • Categorías

    Un gran abanico de categorias, desde informática a comida pasando por otras como diseño, deportes, redes sociales, oficina, tiempo y muchas categorías más.

  • Formatos

    Disponible en EPS, PSD, PNG, SVG y Web Font para que tu puedas incluirlos en tus proyectos web, móvil y escritorio. ¡Ahora disponible en Sketch!

  • Icon Fonts

    Utiliza los iconos como Web Font para crear bonitas y claras interfaces para sitios web de una manera más cómoda.

Personalización

Captain Icon ha trabajado duro para ofrecerte un completo set de iconos gratuito donde puedas personalizar lo que quieras, desde tamaños a colores pasando por formas, nodos y mucho más.

Como Captain Icon conoce exactamente como trabajas, ha creado el set de iconos gratuito tanto para aplicaciones gráficas como para aplicaciones vectoriales.
